# Next.js 시작하기

Next.js는 React 기반의 풀스택 웹 프레임워크입니다. 이 글에서는 Next.js의 핵심 개념과 시작 방법을 알아봅니다.

## 왜 Next.js인가?

Next.js는 다음과 같은 장점을 제공합니다:

- **서버 사이드 렌더링 (SSR)** - 더 나은 SEO와 초기 로딩 속도
- **정적 사이트 생성 (SSG)** - 빠른 페이지 로드
- **API Routes** - 백엔드 API를 같은 프로젝트에서 관리
- **이미지 최적화** - 자동 이미지 최적화와 lazy loading

## 프로젝트 생성

Next.js 프로젝트를 만드는 것은 매우 간단합니다:

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

## App Router vs Pages Router

Next.js 13부터 새로운 **App Router**가 도입되었습니다. 주요 차이점은:

| 기능 | App Router | Pages Router |
|------|------------|--------------|
| 레이아웃 | 중첩 레이아웃 지원 | 제한적 |
| 서버 컴포넌트 | 기본값 | 미지원 |
| 데이터 페칭 | async/await | getServerSideProps |
